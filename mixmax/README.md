# 🎬 MIXMAX

Projeto desenvolvido para a disciplina de Programação Web.

O MIXMAX é um site de filmes com temática retrô/synthwave, permitindo:

- Login de usuário
- Cadastro de conta
- Exibição dinâmica de filmes
- Pesquisa de filmes
- Interface estilizada inspirada nos anos 80

---

# 🚀 Tecnologias utilizadas

- HTML5
- CSS3
- JavaScript
- Vite
- LocalStorage
- ECMAScript Modules (ESM)

---

# ⚙️ Funcionalidades

## Login e cadastro
Os usuários podem criar contas e fazer login utilizando LocalStorage.

## Pesquisa de filmes
A pesquisa é realizada utilizando `filter()` para filtrar os filmes dinamicamente.

## Criação dinâmica de componentes
Os cards de filmes são criados pelo JavaScript utilizando:

- `createElement`
- `appendChild`
- `innerHTML`

## Programação funcional
Foi utilizado:

- `map()` para percorrer e renderizar filmes
- `filter()` para pesquisa dinâmica
